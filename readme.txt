TrustPay Payment Gateway for the CampTix plugin

== Description ==

TrustPay Payment Gateway for the CampTix plugin.

You can read more about TrustPay at [trustpay.eu](http://trustpay.eu/)

**This plugin is third-party stuff and it isn't directly associated with Trust Pay, a.s. or officially supported by their developers.**

Thanks the [Slovak WordPress community](http://wp.sk/) and [Webikon](http://www.webikon.sk/) for the support. For more information about me check out my [personal page](http://bocinec.sk/).

== Installation ==

1. Upload `camptix-trustpay` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to `Tickets -> Setup` in your WordPress admin area to set the currency and activate the TrustPay gateway.
